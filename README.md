# Cryptocurrency-Dashboard
1. Project Title-
   Time Series Analysis with Cryptocurrency
What you built: 
 • Built an interactive Power BI dashboard to analyze and forecast cryptocurrency price trends over time.
Why you built it:
• To monitor historical performance, identify trends and volatility, and support data-driven investment and trading decisions.
How you built it:
• Imported and cleaned historical cryptocurrency price data using Power Query.
• Created time-based measures and KPIs using DAX (moving averages, price change %, volatility).
• Designed interactive visuals (line charts, slicers, date filters, plotly to display charts, trends, and predictions) for dynamic analysis.
Result:
• Delivered clear visualization of price trends and forecasts.
• Enabled users to compare historical performance across time periods.
• Improved insight into cryptocurrency market behavior through interactive analytics.
